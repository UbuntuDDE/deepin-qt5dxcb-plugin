# qt5dxcb-plugin

qt5dxcb-plugin is the Qt platform integration plugin for Deepin Desktop Environment.

## Dependencies

### Build dependencies

* qmake (>= 5.3)
* mtdev
* Qt5 (>= 5.3)
  * Qt5-Core
  * Qt5-Gui
  * Qt5-Widgets
  * Qt5-DBus
  * Qt5-Multimedia
  * Qt5-MultimediaWidgets
  * Qt5-Svg
  * Qt5-OpenGL
  * Qt5-X11extras
  * Qt5-Core-Private
  * Qt5-Widgets-Private

## Installation

### Build from source code

1. Make sure you have installed all dependencies.

2. Build:
```
$ cd qt5dxcb-plugin
$ mkdir build
$ cd build
$ qmake ..
$ make
```

3. Install:
```
$ sudo make install
```
